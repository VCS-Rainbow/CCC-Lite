# 闲凉活动室 - 精简版
> Casual Chill Club - Lite Edition

也可以叫【无忧清爽游乐室】

 - 一个可随意放置的小游戏合辑。
 - 是一个用来堆小游戏的企划
 - 主要有**街机**`arcade`和**随身**`dialog`两种形式。
 - 精简版：内置所需前置。

这个企划现在有两个游戏
 - 2048: 仅完整版，还没移植和升级版本
 - c4: 四子棋。
---
### [ Connect 4 - 四子棋 ]
#### 配置
##### 生成街机
**`function ccc-c4:_place.machine`**
>生成默认街机

**`function ccc-c4:_place.machine.custom_pieces {pieces:["方块id"]}`**
> 生成带自定义棋子的街机

##### 生成控制板
**`place template ccc-c4:control_panel`**
> 可以生成控制板，包括一些基础指令的命令方块。
> 可以自由复制这些命令方块，指令会影响到左下角最靠近命令方块的街机。

##### 增加棋子
玩家可以往左下角的炼药锅中丢入方块物品来增加新的棋子
 - 物品必须为完整方块
 - 使用控制板最上方的指令来清空初始的棋子。
 - 使用控制板也可以把棋子重置回来
 - 棋子是实际存在的方块，所以在生存模式使用时有可能被用于刷物品。不要在生存模式玩这个街机。
#### 玩法
**棋盘** 街机的棋盘为 7*6 大小。
**放置** 点击街机下的展示架以放置棋子，棋子会在该列的最上方生成，随重力掉落至下方。
**轮换** 左下角的炼药锅会显示当前的棋子。每次放置之后都会交替到下一个棋子
**悔棋** 连续放置以轮换到原来的棋子，并手动敲掉放错的棋子方块。
**胜利** 连成横竖斜任一线段即告胜利，并清空棋盘。
